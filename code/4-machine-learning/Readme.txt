# Instruções para tf_object_dectection_1.py

   Este modelo pré-treinado deteta caras humanas baseados no algoritmo
   FOMO desenvolvido pela Edge Impulse. Siga as instruções abaixo para o
   colocar a funcionar com a sua Nicla Vision.

1. Copie os ficheiros fomo_face_dection.txt e fomo_face_dection.tflite
   para a raiz da memória da Nicla Vision.

2. Abra o ficheiro tf_object_detection_1.py no editor OpenMV, conecte-se
   à placa e corra o modelo. No terminal do Open MV deverá encontrar
   descrição das caras encontradas pela câmara, enquanto na janela de
   preview as caras encontradas deverão ser assinaladas por um círculo
   verde.
