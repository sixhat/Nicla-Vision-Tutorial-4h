# Instruções para 1-inicio

   Esta pasta contém duas formas de testar o LED interno da placa Nicla
   Vision. São uma forma rápida de diagnosticar o nosso setup. 

1. Abra o ficheiro 1_blink.py no editor e conecte à Nicla Vision. De
   seguida corra o modelo. O LED deverá começar a piscar azul.

2. Abra o ficheiro 2_blink_all.py no editor e conecte à Nicla Vision. De
   seguida corra o modelo. O LED deverá alternar entre as três cores
   primárias.
